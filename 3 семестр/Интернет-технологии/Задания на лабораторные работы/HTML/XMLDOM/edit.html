Пример

Ниже приведен xml-документ, из которого требуется извлечь Овощи (vegetables). 

<note> 	

	<fruits>
		<product> Виноград </product>
		<product> Лимоны </product>
		<product> Мандарины </product>
	</fruits>

	<vegetables>
		<drink> Сок </drink>
		<drink> Лимонад </drink>
		<product> Капуста </product>
		<product> Морковь </product>
		<product> Баклажаны </product>
	</vegetables>

</note>

Используется метод объекта Document - getElementByTagName(tabName), где tagName - имя получаемого тега (узла). С помощью tag=xmldoc.getElementByTagName("vegetables"), (где tag - имя переменной куда необходимо занести результат! А xmldoc- загруженный xml документ) получаем массив объектов в котором будум содержаться все дочернии узлы. К этим узлам обращение идет через - tag.item(i) , где i - номер элемента в массиве. Длина массива - tag.lenght. Для получения значения конкретного узла, его имя необходимо указывать явно. Например, для получения массива объектов "product" необходимо указать tag=xmldoc.getElementByTagName("product")
