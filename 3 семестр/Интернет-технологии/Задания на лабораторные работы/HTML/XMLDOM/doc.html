Объект Document

Является корневым элементом дерева узлов. Все остальные узлы являются childNodes (дочерними узлами) элемента document. Данные элемент должен быть во всех XML-документах. Перечислим основные свойства и методы объекта Document.

Свойства объекта

documentElement - возвращает корневой элемент документа

Методы объекта

createElement(tagName) - создает элемент с заданным именем тэга
createTextNode(text) - создает текстовый узел, содержащий заданный текст
getElementsByTagName - возвращает заданный узел и все его дочерние узлы в виде списка узлов nodeList
