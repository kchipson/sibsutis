Типы узлов DOM

Узлы в XML DOM подразделяются на несколько типов. В данной таблице представлены основные типы узлов (nodeType), их свойства (nodeValue - значение узла, nodeName - имя узла), а также с помощью свойства nodeTypeString можно получить свойство узла Node Type в виде текстовой строки.

Сводная таблица

nodeType nodeTypeString nodeName nodeValue
1element имя тэга null
2attribute имя атрибута значение атрибута
3text #text содержимое узла
