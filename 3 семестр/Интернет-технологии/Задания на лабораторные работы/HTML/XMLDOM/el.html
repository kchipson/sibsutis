Объект Element

Представляет элементные узлы документа. Если элементный узел содержит текст, то этот текст будет представлен в текстовом узле.

Свойства объекта

tagName - возвращает или устанавливает имя узла

Методы объекта

getAttribute(attributeName) - возвращает значение заданного атрибута
getAttributeNode(attributeName) - возвращает заданный узел-атрибут как объект
getElementsByTagName - возвращает заданный узел и все его дочерние узлы в виде списка узлов nodeList

normalize() - помещает текстовые узлы данного элемента и всех его дочерних узлов в один текстовый узел. Не возвращает ничего

removeAttribute(attributeName) - удаляет значение заданного атрибута. Если атрибут имеет значение по умолчанию, то оно устанавливается
removeAttributeNode(attributeName) - удаляет заданный атрибутный узел. Если атрибутный узел имеет значение по умолчанию, этот атрибут вставляется

setAttribute(attributeName,attributeValue) - вставляет новый атрибут
setAttributeNode(attributeNodeName) - вставляет новый атрибутный узел
