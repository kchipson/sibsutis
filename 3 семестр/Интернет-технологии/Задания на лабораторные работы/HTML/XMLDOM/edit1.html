Пример

Ниже приведен xml-документ, из которого требуется извлечь Овощи (vegetables). 

<note> 	

	<fruits>
		<product> Виноград </product>
		<product> Лимоны </product>
		<product> Мандарины </product>
	</fruits>

	<vegetables>
		<drink> Сок </drink>
		<drink> Лимонад </drink>
		<product> Капуста </product>
		<product> Морковь </product>
		<product> Баклажаны </product>
	</vegetables>

</note>

Вариант обращения к узлам через childNodes и nextSibling

Для получение всех дочерних узлов любого узла можно воспользоваться свойством объекта Node - childNodes, который возвратит нам список всех дочерних узлов (NodeList). Например чтобы извлечь все дочерние узлы корневого элемента можно воспользоваться командой x=xmldoc.documentElement.childNodes.И затем обращаться к дочерним узлам, как x.item(i), где i-номер дочернего узла.
