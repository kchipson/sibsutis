Лабораторные задания по курсу  "Интернет-технологии"

Лабораторная работа N 3.4
Использование методов XML DOM

Задание

Создайте форму, состоящей из:

  • текстового поля для вывода названия факультета;
  • текстового поля для вывода курса;
  • выпадающего списка групп. При выделении группы должны автоматически определяться название факультета и курс и выводится в соотвествующие текстовые поля;
  • выпадающего списка ФИО студентов;
  • коллекции радиокнопок по каждому предмету и семестру
  • кнопки "Выполнить", по нажатию которой должен подсчитываться средний балл по предметам, по выбранным семестрам.
  • кнопки "Отменить", по нажатию которой, производится отмена произведенных вычислений.

Программа должна быть написана в соответствии с концепцией объектно-ориентированного программирования. Все обращения к элементам XML-документа должны производится с помощью XML DOM.

*Рекомендации:
реализация данной лабораторной работы возможна в нескольких вариатах. Приведем самые логичные:
1. создать переменную типа - object, в которую будет занесена вся необходимая информация из XML-документа.
2. при первичной загрузки страницы, извлечь необходимую информацию и разместить ее непосредственно на странице. Далее при изменении состояния выпадающих списков, вновь обращатся к XML-документу и извлекать данные, соответствующие текущим условиям.
