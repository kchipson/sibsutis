Создание объекта:
new Date()
new Date(дата)
new Date(год, месяц, дата[, часы[, минуты[, секунды[,милисекунды]]]])

Объект Date предназначен для хранения даты и времени. Если при создании объекта ему не было приписано значение, то он, по умолчанию, хранит текущую дату и время.

Индексация месяцев в Date начинается с нуля.
