Создание объекта:
function(аргументы){тело_функции}
new Function( аргументы, тело_функции );

Объект function используется для создания новой функции.

Тело function может содержать оператор return. returnпозволяет выйти из любой точки внутри функции и вернуть результирующее выражение.

Следующие примеры демонстрирует использование functionn и return: 


function add(x, y){
   return(x + y);
}
var add2 = new Function("x", "y", "return(x+y)");
add(2, 3);
add2(2, 3);
