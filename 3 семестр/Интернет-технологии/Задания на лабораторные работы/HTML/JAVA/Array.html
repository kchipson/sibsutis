Объект Array, представляет собой массив элементов. Индексация в массиве начинается с нуля. Для обращения к элементам используются оператор [].

Объект Array можно использовать, начиная с версии 1.1 языка JavaScript(Netscape Навигатор 3.0).

Пример работы с объектом Array: 


var my_array = new Array();
for (i = 0; i < 10; i++){
    my_array[i] = i;
}
x = my_array[4];
