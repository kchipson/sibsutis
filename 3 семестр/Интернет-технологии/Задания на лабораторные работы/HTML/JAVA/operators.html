Операторы работы с элементами объекта: .,(),[]

Оформление: объект.элемент_объекта

Оформление: массив_элементов(индекс_элемента)

Оформление: функция[аргументы]

Операторы . и [] возвращают элемент, к которому обратились

Оператор () передает аргументы вызываемой функции.

Арифметические операторы: +,-,*,/,%,++,--

Оформление: Выражение1 +,-,*,/ или % Выражение2

Оформление: Выражение ++ или --

Операторы +,-,*,/,% возвращают соотвественно сумму, разность, произведение двух выражений, результат деления первого выражения на второе, остаток от деления первого выражения на второе.

Операторы ++,-- возвращают соотвественно увеличенное на 1 или уменьшенное на 1 значение выражения.

Оператор изменения знака: -

Оформление: унарный_минус выражение

Оператор "-" изменяет знак выражения на противоположный.

Оператор объединения строк: +

Оформление: Строка1 оператор_объединения Строка2

Оператор + возвращает результат объединения двух строк.
Пусть, например, Строка1 = "Привет ", а Строка2 = "мир.". Тогда, результатом выполнения оператора будет "Привет мир."

Операторы работы с памятью: new,delete

Оформление: new конструктор[аргументы]

Оформление: delete выражение

Оператор new вызывает конструктор объекта и выделяет память под объект.

Оператор delete удаляет объект или массив из памяти, и возвращает true при успешном выполнении операции, иначе возвращает false.

Примеры работы с операторами new и delete 


my_object = new Object;
my_array = new Array();
my_date = new Date("Jan 5 1996");
delete my_оbject;

Оператор typeof

Оформление: typeof выражение

The typeof operator returns type information as a string. There are six possible values that typeof returns: "number," "string," "boolean," "object," "function," and "undefined."

Оператор instanceof

Оформление: объект instanceof класс

Оператор instanceof возвращает true, если объект содержиться в заданном классе, иначе возвращает false. 


function objTest(obj) {
   var i, t, s = "";
   t = new Array();
   t["Date"] = Date;
   t["Object"] = Object;
   t["Array"] = Array;
   for (i in t){
      if (obj instanceof t[i]) s += "obj is an instance of " + i + "\n";
      else s += "obj is not an instance of " + i + "\n";
  }
  return(s);
}
var obj = new Date();
response.write(objTest(obj));

Оператор void

Оформление: void выражение

The void operator evaluates its expression, and returns undefined. It is most useful in situations where you want an expression evaluated but do not want the results visible to the remainder of the script.

Операторы сдвига: <<,>>,>>>

Оформление: Выражение1 оператор_сдвига Выражение2

Рассмотрим следующий пример: 


var temp
temp = -14 << 2

Оператор << сдвигает влево биты в Выражении1 на число, заданное в Выражение2, выставляя справа нули. Поэтому в приведенном примере переменная temp будет равно -56, т.к. -14 (11110010) после свига равно -56 (11001000).

Оператор >> сдвигает вправо биты в Выражении1 на число, заданное в Выражение2, выставляя слева единицы. Поэтому в приведенном примере переменная temp будет равно -4, т.к. -14 (11110010) после свига равно -4 (11111100).

Оператор >>> сдвигает вправо биты в Выражении1 на число, заданное в Выражение2, выставляя слева нули. Поэтому в приведенном примере переменная temp будет равно 60, т.к. -14 (11110010) после свига равно 60 (00111100).

Операторы сравнения: <,>,<=,>=,==,!=,===,!==

Оформление: Выражение1 оператор_сравнения Выражение2

Особенности операторов: <,>,<=,>=,==,!=
Если Выражение1 и Выражение2 были различного типа, то они перед сравнением приводятся к общему типу.

Особенности операторов: ===,!==
Если Выражение1 и Выражение2 были различного типа, то операторы сравнения возвращают значение ложь - false.

Общие свойства: Если оба выражения строкового типа, то они сравниваются лексикографически. Для выражений, не имеющих никакого значения, операторы сравнения возвращают значение ложь - false. Если выражения равны, то операторы сравнения возвращают значение истина - true, иначе ложь - false.

Побитовые операторы: ~,&,^,|

Оформление: Выражение1 &,^ или | Выражение2

Оформление: ~ Выражение

Оператор & побитово просматривает оба выражения, и если соотвествующие биты равны 1, то в результирующем выражении на этом месте будет 1, иначе 0. Пусть, например, выражение1 равно -14 (11110010), а Выражение2 равно 14 (00001110), тогда результирующее выражение будет равно 2 (00000010).

Оператор | побитово просматривает оба выражения, и если соотвествующие биты не равны 0, то в результирующем выражении на этом месте будет 1, иначе 0. Пусть, например, выражение1 равно -14 (11110010), а Выражение2 равно 14 (00001110), тогда результирующее выражение будет равно 254 (11111110).

Оператор ^ побитово просматривает оба выражения, и если соотвествующие биты равны 1 или 0, то в результирующем выражении на этом месте будет 1, иначе 0. Пусть, например, выражение1 равно -14 (11110010), а Выражение2 равно 14 (00001110), тогда результирующее выражение будет равно 3 (00000011).

Оператор ! побитово просматривает выражение, и если бит равен 0, то в результирующем выражении на этом месте будет 1, иначе 0. Пусть, например, выражение равно -14 (11110010), тогда результирующее выражение будет равно 13 (00001101).

Логические операторы: !,&&,||

Оформление: Выражение1 && или || Выражение2

Оформление: ! Выражение

Оператор && возвращает значение истина - true, если оба выражения имеют значение истина - true. Оператор && возвращает значение ложь - false, если выражения являются пустыми строками, нулевыми числами, равны NULL или их значение неизвестно.

Оператор || возвращает значение истина - true, если оба выражения не имеют значение ложь - false. Оператор || возвращает значение ложь - false, если выражения являются пустыми строками, нулевыми числами, равны NULL или их значение неизвестно.

Оператор ! возвращает значение истина - true, если выражение имеет значение ложь - false. Оператор ! возвращает значение ложь - false, если выражение является ненулевым числом. Если выражение, является объектом(строкой), то оно перед выполнением оператора переводится в число.

Оператор условия: ?:

Оформление: условие ? операторы1 : операторы2

Оператор ?: проверяет заданное условие и если оно имеет значение истина - true, то выполняет операторы1, иначе выполняет операторы2.

Использование оператора ?: показано в следующем примере: 


var now = new Date();
var greeting = "Good" + ((now.getHours() > 17) ? " evening." : " day.");

Операторы присвоения: =,+=,&=,|=,^=,/=,<<=,%=,*=,>>=,-=,>>>=

Оформление: Переменная оператор_присвоения Выражение

Оператор = присваивает Переменной, стоящей слева Выражение, стоящее справа.

Операторы +=,&=,|=,^=,/=,<<=,%=,*=,>>=,-=,>>>= сперва выполняют соотвествеющие операции, стоящие перед знаком "=" над значениями Переменной и Выражения. Результат полученный после выполнения этих операций присваивается Переменной, стоящей слева от оператора.

Оператор перечисления: ,

Оформление: Выражение1 оператор_перечисления Выражение2

Оператор "," предназначен для разделения выражений.

Использование оператора "," показано в следующем примере: 


for (i = 0; i < 10; i++, j++){
  k = i + j;
}
