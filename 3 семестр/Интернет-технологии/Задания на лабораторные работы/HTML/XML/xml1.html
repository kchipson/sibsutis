Представление данных XML

Задание

Преобразовать предложенные XML данные средствами таблиц XSL в соответствии с вариантом.

  1. Создать таблицу. Данные в таблице должны быть отсортированы в начале по количеству спутников, затем по названию(в порядке возрастания). Выделить одним цветом данные о планетах, радиусом меньше 10000км и другим данные об остальных. В первом столбце должен располагаться номер строки.

  2. Поместить данные о планетах, расположенных в области, ограниченной астероидным поясом (Меркурий-Марс) в одну таблицу (таблица должна быть отсортирована по диаметру), данные об остальных планетах поместить в другую таблицу(таблица должна быть отсортирована по количеству спутников). В обоих таблицах выделить столбец значений массы относительно Земли.

  3. Поместить данные о планетах, имеющих спутники в таблицу. Данные в таблице отсортировать по количеству спутников. Разметить четные строки таблицы одним цветом, нечетные другим. Данные о планетах, не имеющих спутников представить в виде списка определений.

  4. Создать таблицу таким образом, чтобы свойства планет располагались по строкам а названия планет по столбцам. Строку с названиями планет выделить другим цветом.
*Исходные данные должны быть сохранены в файл с расширением xml. Преобразования должны осуществляться при подключении к исходному файлу файла таблиц стиля xsl.
