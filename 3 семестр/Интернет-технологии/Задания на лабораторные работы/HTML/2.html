СSS - Cascading Style Sheets - данную технологию можно перевести как "Каскадные Таблицы Стилей" или"Каскадный стиль документа" CSS - это язык содержащий набор свойств для определения внешнего вида документа. С его помощью автор HTML документа имеет полный контроль над стилем и расположением каждого элемента (или объекта) WWW страницы. Это гораздо проще использования обычного набора HTML команд (tag'ов),особенно если стиль представления объектов нужно использовать многократно.

Существует три вида таблиц стилей: Внутренние таблицы стилей, Глобальные таблицы стилей и Связанные таблицы стилей.

Внутрение таблицы стилей (Inline Style Sheets) мало чем отличаются от HTML деклараций.
Глобальные (Global Style Sheets) определяют стиль элементов во всем документе.
Связанные (Linked Style Sheets) могут быть использованы для нескольких документов и хранятся во внешнем файле.
