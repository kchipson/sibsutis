Расчетно-Графическое Задание по курсу "Введение в Интернет"
Построение сайта с использованием CMS (Content Management System)

  • Тема сайта — любая.

  • Сайт должен состоять из нескольких (не менее 5) страниц.

  • В Сайте должны использоваться блочные структуры, фреймы, элементы JavaScript и пр.

  • Навигация по сайту всегда доступна через отдельный раздел меню. При клике мышкой  — открывается выбранная страница.

  • В оглавлении сайта каждый пункт (гиперссылка) должен быть оформлен картинкой.

  • При наведении мышки на гиперссылку картинка должна меняться (применить JavaScript).
