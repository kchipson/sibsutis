Предметный указатель по языку HTML

Здесь даны описания и примеры наиболее употребляемых спецификаций языка HTML 3.2.

Команда
Tag 		Описание
!--...-- Комментарии. Текст, который игнорируется browser'ом.
!DOCTYPE Описание версии языка HTML, используемой в документе (первая команда документа).
A Описание ссылок (anchor). Атрибут HREF= задает гиперссылки. Атрибут NAME= задает метки внутри документа.
ADDRESS Форма для написания адреса (например, адреса электронной почты).
APPLET Определение Java приложения (applet). См. OBJECT.
AREA Задание ссылок на рисунке (активные рисунок или карта).
B Задание полужирного шрифта для текста. См. STRONG.
BASE Задание базового адреса URL для документа.
BASEFONT Задание базового типа и размера шрифта для документа.
BGSOUND Задание фонового звука для документа.
BIG Увеличение размера шрифта.
BLOCKQUOTE Абзацное выделение текста (quotation).
BODY Задание начала и конца тела документа. См. HEAD.
BR Жесткий конец строки (переход на следующую строку).
CAPTION Задание заголовка в таблице.
CENTER Центрирование.
CITE Задание цитаты.
CODE Вывод текста шрифтом фиксированной ширины.
COL Задание свойств столбцов в таблице.
COLGROUP Задание свойств столбцов в таблице, объединенных в группу.
COMMENT Задание комментария. Текст, заключенный внутри команды игнорируется просмотрщиком.
DD Задание данных в списке определений. См. DL, DT.
DFN Вывод определений (теорем).
DIR Задание списка описаний (directory - см. Списки).
DIV Относительное размещение текста в документе.
DL Объявление списка определений. См. DT, DD.
DT Задание определения для списка определений. См. DL, DD.
EM Задание шрифта выделений (как правило курсив).
EMBED Задание вложенных объектов (objects). См. OBJECT.
FONT Задание шрифта (стиль, размер, цвет -- style, size, color).
FORM Задание формы для ввода данных для сервера. См. INPUT.
FRAME Задание нескольких независимых окон (frames) в одной странице документа. См. FRAMESET.
FRAMESET Определение параметров отдельных окон на странице. См. FRAME.
Hn Задание заголовков (N принимает значения от 1 до 6).
HEAD Задание параметров HTML документа (document heading).
HR Горизонтальная линия.
HTML Обозначения файла как HTML документа (вторая команда в документе).
I Выбор шрифта курсив (italic).
IMG Вставка графического файла (рисунка).
INPUT Задание элементов ввода в команде FORM. См. FORM.
ISINDEX Контроль необходимых полей данных для поиска.
KBD Определяет текст для ввода с клавиатуры (выводится полужирным шрифтом фиксированной ширины).
LI Определение элемента списка (List item). См. UL, OL.
LINK Определение связей между документами. Элемент раздела HEAD.
LISTING Вывод текста шрифтом фиксированной ширины.
MAP Определение ссылок на картинке (карте).
MARQUEE Вывод текста со сролингом (поддерживается только MS Internet Explorer).
MENU Список (см. Cписки).
META Задание информации о документе. Элемент раздела HEAD.
NOBR Отключение режима переноса строк.
NOFRAMES Задание раздела документа для просмотрщиков не поддерживающих многооконность (frames).
OBJECT Вставка OLE Control.
OL Определение нумерованного списка. См. UL, LI, а также раздел Cписки.
OPTION Определение метки (см. FORM, SELECT, а также раздел Формы).
P Задание нового "параграфа" (начало и конец абзаца).
PARAM Определение свойств OBJECT'а.
PLAINTEXT Вывод текста шрифтом фиксированной ширины, в режиме игнорирования команд.
PRE Вывод текста так как он напечатан — со всеми пробелами и концами строк (допускаются ссылки и команды).
S Зачеркнутый текст (синоним команды STRIKE).
SAMP Вывод текста, определяющего примеры (см. CODE).
SCRIPT Подключение элементов script'a.
SELECT Определяет список меток для выбора пользователем (см. FORM а также раздел Формы).
SMALL Уменьшение размера шрифта.
SPAN Дополнительная команда для команды STYLE.
STRIKE Зачеркнутый текст. См. S.
STRONG Выделение текста (как правило полужирным шрифтом). См. B.
SUB Нижний индекс (subscript).
SUP Верхний индекс (superscript).
TABLE Создание таблицы. См. TH, TR, и TD для определения строк и столбцов таблицы.
TBODY Задание тела таблицы.
TD Задание клетки таблицы.
TFOOT Определение нижней части таблицы (footer).
TH Определение строк и столбцов верней части таблицы (heading).
THEAD Задание верней части таблицы (header).
TEXTAREA Создание бокса для ввода текста (см. также раздел Формы).
TITLE Задание заголовка документа (используется для поиска).
TR Задание строки в таблице.
TT Равноширинный текст.
U Подчеркнутый текст.
UL Ненумерованный список. См. LI, а также раздел Cписки.
VAR Ввод переменных. Печатается малым шрифтом фиксированной ширины.
WBR Вставка мягкого конца строки в блок, определяемый командой NOBR.
XMP Ввод примеров. Печатается текстом фиксированной ширины.
