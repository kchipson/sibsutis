Метод ceil

Возвращает ближайшее целое число, большее или равное аргументу.

Синтаксис

 Math.ceil (number)

number - любое числовое выражение.

Метод

Math

Примеры

  
// Показывает значение 46 
document.write (" ceil 45.95 - " + Math.ceil (45.95))

// Показывает значение -45 
document.write ("The Ceil -45.95 - " + Math.ceil (-45.95))

    Метод charAt

    Возвращает символ из строки.

    Синтаксис

     StringName.charAt (index)

    StringName - любая строка или свойство существующего объекта.
    Индекс - любое целое число от 0 до stringName.length - 1, или собственности существующего объекта(цели).

    Объект checkbox

    Измененный в Navigator 3.0

    Элемент формы, который пользователь устанавливает в состоянии on или off посредством щелчка левой кнопкой миши.

    Синтаксис

    Чтобы определить объект checkbox, используйте стандарт HTML синтаксис с использованием обработчика событий onClick:
    <INPUT TYPE="checkbox" NAME="checkboxName" VALUE="checkboxValue" [CHECKED] [onClick="handlerText"]> textToDisplay
    NAME = "checkboxName" определяет имя объекта checkbox. Вы можете обращаться к этому значению, используя свойство имени.
    VALUE = "checkboxValue" определяет значение, которая возвращена серверу, когда checkbox выбран, и форма подчинена. Это значение по умолчанию on. Вы можете обращаться к этому значению, используя свойство значения.
    CHECKED определяет состояние флажка. Вы можете обращаться к этому значению, используя свойство defaultChecked.
    TextToDisplay определяет значение, которое показано на экране.

    Использование свойству объекта и методы checkbox: 

      
1. CheckboxName.propertyName 
2. CheckboxName.methodName (parameters) 
3. FormName.elements [index] .propertyName 
4. formName.elements[index].methodName(parameters)
    CheckboxName - значение атрибута NAME объекта checkbox.
    FormName является либо значением атрибута NAME объекта формы либо элемента в массиве form.
    index - целое число, представляющее объект checkbox в форме.
    PropertyName - один из свойств, описанных ниже.
    MethodName - один из методов, описанных ниже.

    Свойство

  • form

    Описание

    Объект checkbox в форме выглядет следующим образом:

    Объект checkbox - элемент формы и должен быть определен внутри тега HTML <FORM>.

    Свойства и методы флажков имеют множество применений.
    Используйте свойство checked объект checkbox для проверки текущего состояния флажка: установлен(true) или нет(false). Если атрибут CHECKED был использован в определении флажка, свойство defaultChecked также возращает true.

1. CheckboxName.checked 
2. RadioName [index] .checked

    RadioName - значение атрибута NAME объекта radio.
    index - целое число, представляющее переключатели в объекте radio.

    checkbox, radio

    Метод click

    Эмулирует щелчок на объекте левой кнопкой мыши.

    Синтаксис

      
1. ButtonName.click () 
2. RadioName [index] .click () 
    
3. CheckboxName.click ()

    ButtonName является либо значением атрибута NAME кнопки reset либо объект submit либо элемент в массиве elements.
    RadioName - значение атрибута NAME объекта radio либо элемент в массиве elements.
    index - целое число, представляющее переключатель в объекте radio.
    CheckboxName является либо значением атрибута NAME объекта checkbox либо элемент в массиве elements.

    Метод

    button, checkbox, radio, reset, submit

    Описание

    Эффект щелчка зависит от типа элемента формы.

    Воздействие метода click на элементы формы:

  • Для button, выполняет то же самое действие, что и нажатие кнопки.
  • Для radio, выбирает переключатель.
  • Для checkbox, устанавливает флажок и присваивает ему значение on.

    Примеры

    Следующий пример переключает состояние выбора первого переключателя в musicType объекте radio в форме musicForm: 
     Document.musicForm.musicType [0] .click ()

    Следующий пример переключает состояние выбора переключателя newAge в форме musicForm: 
     Document.musicForm.newAge.click ()

    Метод confirm

    Показывает диалоговое окно

    JavaScript Confirm с указанным сообщением и BATTON "OK" и "Cancel".

    Синтаксис

    confirm ("message")

    message - любая строка или свойство существующего объекта.

    Метод

    window

    Описание

    Используйте метод confirm, чтобы пользователя подтвердил принятое решение. Аргумент message определяет сообщение, которое побуждает пользователя для решения. Подтверждение возвращает методу, true, если пользователь выбирает "OK" или false если пользователь выбирает "Cancel".

    confirm - метод объекта окна, вы не должны определять windowReference, когда Вы вызываете его. Например, windowReference.confirm () ненужен.

    Примеры

    Этот пример использует метод confirm в функции confirmCleanUp (), чтобы подтвердить, что пользователь действительно хочет выйти. Если пользователь выбирает OK, то функция cleanUp() закрывает приложение.
    function confirmCleanUp() { if (confirm("Are you sure you want to quit this application?")) { cleanUp() } }
    Вы можете вызвать функциею confirmCleanUp () в обработчике событий onClick кнопки формы, как показано в следующем примере:
    <INPUT TYPE="button" VALUE="Quit" onClick="confirmCleanUp()">

    Свойство cookie

    Значение строки cookie, которая содержит небольшое количество информации, сохраненной Navigator в файле cookies.txt.

    Синтаксис

     Document.cookie

    Свойство

    document

    Описание

    Значение, хранимое в свойстве cookie, может быть извлечено с помощью методов charAt, indexOf, и lastIndexOf.

    Вы можете управлять свойством cookie в любое время.

    Примеры

    Следующая функция использует свойством cookie, чтобы делать запись напоминания для пользователей заявления(применения). " Истекает = " компонента, устанавливает дату истечения срока для печенья, так что это сохраняется вне текущего сеанса окна просмотра. Формат даты должен быть 

      
Wdy, DD-Mon-YY HH:MM:SS GMT
    Где Wednesday - имя дня недели, DD - днь месяца, Mon - месяц, YY - последние два числа года, и HH, MM, и SS - соответственно часы, минуты, и секунды.

    Это тот же формат даты какой возвращяет Date.toGMTString, за исключением:

    • Черточек между днем, месяцем, и годом
    • Год состоит из двух-однозначных чисел для cookies.

    Например, имеет силу истечения даты cookies:
    expires=Wednesday, 09-Nov-99 23:12:40 GMT

    function RecordReminder(time, expression) { // Делает запись <b>cookies</b> формы "@=" К отображению // из <T> В миллисекундах начиная с периода, // возвращенный Date.getTime (), на закодированное выражение, // <E> ( Закодированный, чтобы содержать никакое незаполненное пространство, точку с запятой, // или цифры(знаки) запятой) document.cookie = "@" + time + "=" + expression + ";" // урегулируют истечение печенья время к одному дню // вне напоминания, время document.cookie += "expires=" + cookieDate(time + 24*60*60*1000) // cookieDate - функция, которая форматирует дату согласно спецификациям печенья }

    Когда пользователь грузит страницу, которая содержит эту функцию, другая функция использует indexOf ("@") и indexOf ("=") чтобы определить дату и время, сохраненную в печеньи.

    Метод cos

    Возвращает косинус своего аргумента.

    Синтаксис

     Math.cos (number)

    number - числовое выражение, представляющее размер угла в радианах.

    Метод

    Math

    Описание

    Метод cos возвращает числовое значение в диапозоне между -1 и 1.

    Примеры

    // Показывает значение 6.123031769111886e-017 
document.write (" косинус PI/2 радиан,  " + Math.cos (Math. PI/2))
// Показывает значение -1 
document.write (" косинус PI радиан " + Math.cos (Math. PI))

// Показывает значение 1 
document.write (" косинус 0 радиан - " + Math.cos (0))
