9.3. Пространство имен

 
int fd;
fd = open ("/bin/true", ...);
read (fd, ...);
close (fd);


1. ConnectAttach procnto, MsgSend -> "open /bin/true"
       <- "ask Сервер B" (самое длинное совпадение)
       <- "ask Сервер A" (короткое совпадение)

2. ConnectAttach Сервер B, MsgSend -> "open true"
       <- "нет такого файла"

3. ConnectAttach Сервер A, MsgSend -> "open bin/true"
       <- "OK"
