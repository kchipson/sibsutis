8.8. Задача

Три нити T1, T2, T3 с номерами соответственно 1, 2, 3 работают на одном процессоре, начальный приоритет равен 10, алгоритм планирования FIFO. Определить последовательность выводимых символов. 
void* T1 (void* foo)
  {
  sem_wait (&sem);
  putchar ('A');
  setprio (0, 11);
  pthread_barrier_wait (&bar);
  putchar ('I');
  sem_wait (&sem);
  putchar ('I');
  sem_post (&sem);
  pthread_join (2, 0);
  putchar ('T');
  }

void* T2 (void* foo)
  {
  pthread_barrier_wait (&bar);
  putchar ('M');
  sem_post (&sem);
  putchar ('R');
  sched_yield ();
  putchar ('U');
  }

void* T3 (void* foo)
  {
  putchar ('V');
  sem_post (&sem);
  putchar ('L');
  sched_yield ();
  putchar ('D');
  pthread_barrier_wait (&bar);
  sem_wait (&sem);
  putchar ('P');
  setprio (0, 9);
  putchar ('I');
  sched_yield ();
  putchar ('N');
  }

Перед запуском нитей выполнена сдедующая инициализация: 

  sem_init (&sem, 0, 0);
  pthread_barrier_init (&bar, 0, 3);
