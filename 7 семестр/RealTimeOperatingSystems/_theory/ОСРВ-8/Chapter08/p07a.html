8.7. Переменные условий (condvars)
Системные вызовы (POSIX)
#include <pthread.h>

pthread_cond_t cond; описание
pthread_cond_init (&cond, 0); инициализация
0 = тип по умолчанию

pthread_cond_wait (&cond, &mutex); ожидание
pthread_cond_broadcast (&cond); разблокирует все нити,
ожидающие cond

Техника использования: 

pthread_mutex_lock (&mutex);
...
while (!condition) pthread_cond_wait (&cond, &mutex);
...
pthread_mutex_unlock (&mutex);

Сигнал выводит нить из ожидания, errno = OK.
