8.6. Барьеры
Системные вызовы (POSIX)
#include <pthread.h>

pthread_barrier_t barrier; описание барьера
pthread_barrier_init (&barrier, 0, int count); инициализация барьера
0 = тип по умолчанию

pthread_barrier_wait (&barrier); ожидание барьера
(д.б. вызвана count раз)

После перехода через барьер барьер переводится в начальное состояние.

pthread_barrier_wait() не прерывается сигналом.
