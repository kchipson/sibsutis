8.1. КС и повторная входимость функций
Повторная входимость

Функция называется повторно входимой (реентерабельной), если она допускает повторные вызовы во время своей работы.

Пример. Функция установки цвета символов и фона для терминала ANSI: 

static char cmd[] = "\033[=nF\033[=nG";
void setcolor (int f, int b)
  {
  cmd[3] = f + '0', cmd [8] = b + '0';
  write (1, cmd, 10);
  }

Данная функция не реентерабельна (на уровне нитей).
