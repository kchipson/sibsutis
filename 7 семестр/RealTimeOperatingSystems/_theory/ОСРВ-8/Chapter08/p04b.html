8.4. Мьютексы
Идея реализации (x86)

Команды процессора: 


    bts $bit, mem          mem[bit] -> flag C
                           mem[bit] <- 1
                           
    btr $bit, mem          mem[bit] -> flag C
                           mem[bit] <- 0

    bt  $bit, mem          mem[bit] -> flag C
