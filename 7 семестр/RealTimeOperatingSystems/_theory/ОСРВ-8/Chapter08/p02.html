8.2. Семафоры

#include <semaphore.h>

Семафор -- объект типа sem_t. Содержит поле count = 0, 1, 2, ...

sem_init (sem_t* sem, int pshared, unsigned value);

Инициализация семафора.   Параметры:

sem
указатель на семафор
pshared = 0
семафор используется нитями в одном процессе
pshared = 1
семафор используется разными процессами
(должен находиться в разделяемой памяти)
value
начальное значение семафора

sem_destroy (sem_t* sem);

Разрушение семафора.
