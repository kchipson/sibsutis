7.6. Функции таймерной поддержки (POSIX)

 

struct itimerspec { struct timespec it_value, it_interval; };

timer_settime (int timerid, int flags, struct itimerspec* ts, 0);

Запускает (останавливает) таймер timerid.

Время срабатывания таймера указывается с помощью параметра ts. Поле it_value содержит значение времени при первом запуске (секунды и наносекунды). Это время может быть относительным или абсолютным, что определяется параметром flags (0 или TIMER_ABSTIME).

Если it_value = 0, то таймер останавливается.

Поле it_interval содержит интервал перезапуска таймера. Если it_interval = 0, то перезапуски не планируются.

При повторном вызове timer_settime() таймер "перезаряжается" на новое время срабатывания.

Когда таймер срабатывает, то генерируется событие, прописанное при его создании.
