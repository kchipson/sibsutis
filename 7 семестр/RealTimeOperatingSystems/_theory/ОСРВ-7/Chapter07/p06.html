7.6. Функции таймерной поддержки (POSIX)

 

#include <time.h>

clock_nanosleep (int clockid, int flags,
                 const struct timespec* rqtp, struct timespec* rmtp);

Приостанавливает выполнение текущей нити на указанное время.

Параметр clockid задает тип часов, к которым привязывается (неявно создаваемый) таймер.

Параметр flags м.б. 0 либо TIMER_ABSTIME.

С помощью rqtp задается требуемый временной интервал (flags=0) или время, до наступления которого нить будет спать (flags=TIMER_ABSTIME).

Указатель rmtp (м.б. 0) показывает, куда нужно записать оставшееся время сна, если сон был прерван сигналом.
