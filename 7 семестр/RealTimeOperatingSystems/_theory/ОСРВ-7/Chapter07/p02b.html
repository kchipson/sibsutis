7.2. Часы стандарта POSIX
Системные функции для работы с часами 

#include <time.h>

struct timespec {
  int tv_sec, tv_nsec };
int time (0);
возвращает текущее системное время в секундах

clock_gettime (int clockid, struct timespec* ts);
считывает показание часов clockid в структуру по указателю ts

clock_getres (int clockid, struct timespec* ts);
записывает величину тика часов clockid в структуру по указателю ts

clock_settime (int clockid, struct timespec* ts);
устанавливает новое время из структуры по указателю ts в часы clockid.
