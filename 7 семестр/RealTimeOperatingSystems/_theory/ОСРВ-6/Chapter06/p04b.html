6.4. Обработка прерываний: системные функции

 

int InterruptMask (int intr, int isrid);

Маскирует (запрещает) прерывание для указанного обработчика.

Возвращает счетчик маскирований.

Неявно вызывается при входе в обработчик и при выполнении InterruptDetach(). 





int InterruptUnmask (int intr, int isrid);




 Разрешает (размаскирует) прерывание для указанного обработчика.


 Возвращает счетчик маскирований.


 Неявно вызывается при выходе из обработчика и при выполнении
InterruptAttach().









 При генерации события посредством
InterruptAttachEvent()
прерывание остается замаскированным, пока нить не вызовет
InterruptUnmask().
