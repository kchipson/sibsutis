3.6. Нити
POSIX threads 1003.1
#include <pthread.h>
Создание нити
pthread_create (int* tid, 0, void* (*func)(void*), void* arg); 





tid  куда положить номер нити, м.б. 0

0  вместо указателя на структуру с атрибутами нити

func    указатель на функцию, которая запускается в виде нити

  (первичная функция нити)

arg  аргумент, который будет передан в func








Завершение нити




(1) Выход из первичной функции нити (с возвратом статуса завершения)

(2) pthread_exit (void* stat);

(3) pthread_cancel (int tid);
