3.6. Нити
Динамическое планирование нитей

pthread_setschedparam (int tid, int alg, const struct sched_param *param); 




 Устанавливает для нити новые параметры планирования (приоритет).






Изменение приоритета текущей нити




#include <sched.h>

setprio (0, int prio);      (QNX)






Динамическая карусель




#include <sched.h>

sched_yield ();


 Ставит вызвавшую нить в конец очереди текущего приоритета.
