3.7. Разделяемые объекты и библиотеки

Инфраструктура для PIC

ld создает две таблицы:
_GLOBAL_OFFSET_TABLE_    (GOT)
_PROCEDURE_LINKAGE_TABLE_  (PLT)

GOT размещается в сегменте .data, ее можно модифицировать.

PLT размещается в сегменте .text, она не изменяется.

GOT содержит в себе адреса всех глобальных процедур и данных, с которыми работает разделяемый объект. Эти адреса настраиваются независимо для каждого процесса.

PLT содержит точки входа для вызова глобальных процедур.

Разумеется, в каждом объекте имеется также таблица имен символов.
