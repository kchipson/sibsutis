3.5. Построение отображений
#include <sys/mman.h>
void* mmap (void* addr, int len, int prot, int flags, int fd, int offset );

Отображает область виртуального адресного пространства на файл или физическую память.

addr, len
адрес и длина области. Рекомендуется задавать addr = 0.
prot
режим доступа к отображаемой области:
PROT_READ   разрешение чтения
PROT_WRITE разрешение записи
PROT_EXEC   разрешение выполнения
PROT_NONE   запрет доступа
PROT_NOCACHE  запрет кеширования
