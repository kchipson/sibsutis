3.7. Разделяемые объекты и библиотеки

Инфраструктура для PIC

Для работы с таблицами GOT и PLT программа ld предлагает следующие конструкции идентификаторов:

name@GOT
индекс в таблице GOT для адреса глобальной переменной или функции с именем name
name@GOTOFF
смещение от начала таблицы GOT до метки name внутри разделяемого объекта
name@PLT
адрес точки входа таблицы PLT для глобальной функции с именем name

При работе с таблицами предполагается, что регистр %ebx указывает на начало таблицы GOT.
