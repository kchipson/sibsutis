3.7. Разделяемые объекты и библиотеки

Загрузка динамической библиотеки

#include <dlfcn.h>

void* dlopen (const char *path, int mode);

Загружает динамическую библиотеку и отображает ее в адресное пространство процесса.

path
путь к библиотеке
если указано только имя, то библиотека ищется в каталогах LD_LIBRARY_PATH
mode
управление режимом разрешения имен и видимостью
mode = RTLD_NOW     все ссылки настраиваются при загрузке библиотеки
mode = RTLD_LAZY    ссылки настраиваются по мере их использования

Функция возвращает handle, использующийся для доступа к библиотеке. 


void* dlsym (void *handle, const char *name);

Возвращает адрес функции или др. объекта с именем name в библиотеке. 


dlclose (void *handle);

Закрывает библиотеку.
