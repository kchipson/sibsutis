3.2. Процессы
Запуск внешних программ (fork-exec)

#include <process.h>
execlp (const char *file, const char *arg0, const char *arg1, ..., 0);

Замещает образ текущего процесса образом из файла
(pid и дескрипторы открытых файлов сохраняются).

#include <unistd.h>
pipe (int fd[2]);

Создает трубу (pipe) для обмена данными между процессами.
В fd[0] записывается файловый дескриптор для чтения из трубы,
в fd[1] -- файловый дескриптор для записи в трубу.

#include <unistd.h>
dup2 (int fd1, int fd2);

Дупликация дескриптора открытого файла fd1 в другой дескриптор fd2
(если под fd2 был открытый файл, то он закрывается).
