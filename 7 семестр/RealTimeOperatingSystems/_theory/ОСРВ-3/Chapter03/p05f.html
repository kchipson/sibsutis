3.5. Построение отображений
Взаимодействие с физическими устройствами

(1) Устройства с двухпортовой памятью (графический адаптер)

ptr = mmap (0, len, prot+PROT_NOCACHE, MAP_SHARED + MAP_PHYS, -1, PA);

(2) Устройства с DMA (сетевой адаптер)

ptr = mmap (0, len, prot+PROT_NOCACHE, MAP_SHARED + MAP_PHYS + MAP_ANON, -1, 0);

Физический адрес области можно узнать с помощью функции mem_offset(...).
