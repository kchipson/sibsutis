1.6. Интерпретатор команд - Shell
Перенаправление ввода-вывода

Программа, запущенная из командной строки, получает три открытых потока ввода-вывода:

Название потокаФайловый дескриптор  Указатель на структуру FILE
Стандартный ввод0stdin 
Стандартный вывод1stdout
Стандартный вывод ошибок  2stderr

По умолчанию все потоки ассоциированы с терминалом, однако, они могут быть перенаправлены:

> file     stdout →  file
>> file   то же самое в режиме добавления к файлу
2> file    stderr →  file
< file      stdin ←  file
prog1 | prog2    stdout(prog1) →  stdin(prog2)
