1.6. Интерпретатор команд - Shell
Интерпретаторы:    Bourne, Korn, C, Bourne-Again, ...
Программа на языке интерпретатора - скрипт.
Редактирование командной строки
↑↓    вызов ранее введенных команд
Ctrl-U    аннулировать команду
EscEsc    завершить частично сделанный ввод команды или файла
Esc?    выдать список команд или файлов, соответствующих частично набранному слову
Переменные
Определение переменной var:   var=value
Использование потока вывода команды:   var=`command`
Получение значения переменной:   $var
Встроенные переменные:
$1, $2, ..., $9   позиционные параметры скрипта
$?   код завершения программы (0=OK)
Стандартные переменные:   PATH, LOGNAME, HOSTNAME, HOME, PS1, ...
Объявление var как глобальной переменной:   export var
Вывод всех определенных переменных:   set
Удаление переменной var:   unset var
