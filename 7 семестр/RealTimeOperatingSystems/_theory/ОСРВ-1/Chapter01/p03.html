1.3. Распечатка каталога - утилита ls
ls [opt] [dir|file]
Опции:
-a   - включить имена, начинающиеся с "."
-L   - раскрывать символические ссылки
-d   - выдать информацию о самом каталоге, не раскрывая его
-C   - вывод в несколько колонок (для less/more)
-l (ell)  - "длинный" формат распечатки:
txxxxxxxxx n user group size date & time name
