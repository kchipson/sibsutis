1.10. Вывод информации о процессах
ps

Выводит информацию о процессах пользователя в формате PID TTY TIME NAME
sin

Выводит информацию о всех процессах в формате NAME PID CODE DATA STACK STACK_MAX TIME(ms)
pidin

Выводит информацию о всех процессах в формате PID TID NAME PRIO STATE BLOCKED

pidin mem

Выводит информацию о памяти и разделяемых (динамически загружаемых) объектах
