1.6. Интерпретатор команд - Shell
Команда test
test выражение   возвращает 0, если выражение истинно, 1 в противном случае
Некоторые выражения
-f file   файл file существует
-d dir   каталог dir существует
str1 = str2   две строки идентичны
str1 != str2   две строки различны
i1 -eq i2   числа i1 и i2 равны
i1 -ne i2   числа не равны
i1 -lt i2   i1 меньше i2
i1 -le i2   i1 меньше или равно i2
Операции с выражениями
! выражение   отрицание выражения
выражение1 -a выражение2   логическое И
выражение1 -o выражение2   логическое ИЛИ
