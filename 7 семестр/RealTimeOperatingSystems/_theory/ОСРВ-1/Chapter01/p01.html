1.1. Каталоги и файлы
Типы файлов:
  • регулярный файл
  • каталог
  • символическая связь
  • специальный файл устройства
  • сокет
  • ...
/корень пространства имен
/dir1/dir2/.../file  абсолютный путь к файлу
dir1/dir2/.../fileпуть к файлу относительно рабочего каталога
.ссылка на текущий каталог
.. ссылка на вышестоящий каталог



Утилиты и команды:



pwdпечать рабочего каталога


cd dirустановка рабочего каталога


mkdir dir  создание нового каталога


touch file  создание пустого файла
