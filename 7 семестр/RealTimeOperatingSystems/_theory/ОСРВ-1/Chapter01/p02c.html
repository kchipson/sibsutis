1.2. Редактор vi

Режим input

Ввод текста. Символы ←↑→↓, Enter, Backspace действуют как обычно.

Режим ex

Команды вводятся в строковом режиме.

Основные команды:
:r файл   - считать файл, вставить содержимое после курсора
:w файл  - записать весь текст в указанный файл
:x       - записать в файл и выйти из редактора
:q!     - выйти из редактора без записи в файл
