1.11. Разработка программ на Си

GNU C (gcc) версии 2.95.3 и 3.3.1

cc [opt] file1 file2 ...

Компилирует указанные файлы. Автоматически вызывает нужные программы, исходя из расширений имен файлов.

Расширения:
.c    программа на Си
.cpp    программа на Си++
.s    программа на ассемблере as
.o    объектный файл
.a    статическая библиотека
.so    динамическая библиотека
Опции:
-o name    задает имя выходного (исполняемого) файла (если опция не указана, получается a.out)
-O    оптимизация
-S    сделать .s файл
-l lib    подключить библиотеку lib (стандартная библиотека libc.so подключается автоматически)
