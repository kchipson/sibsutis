1.8. Пользователи системы
База данных пользователей (3 текстовых файла):
/etc/passwd    информация о пользователях
/etc/group       информация о группах
/etc/shadow    информация о паролях

Формат записи в файле /etc/passwd:

logname : x : uid : gid : real name : home dir : shell

Формат записи в файле /etc/group:

groupname : x : gid : [user1, user2, ...]

Основные утилиты:

passwd    изменение пароля
login       вход (под другим именем)
logout    выход
newgrp [group]    изменение группы пользователя
