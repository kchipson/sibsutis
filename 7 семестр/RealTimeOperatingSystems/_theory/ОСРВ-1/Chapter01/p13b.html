1.13. Вывод информации и временная приостановка
Приостановка программы

#include <unistd.h>
int usleep (int usec);

Приостанавливает выполнение программы на usec микросекунд (1с = 1000000мкс).

Функции ввода-вывода:
1) с внутренней буферизацией через структуру FILE
fwrite, fread, ..., printf, ...
2) без буферизации через fd
write, read, ...
#include <unistd.h>
int write (int fd, const void * buf, int n);

fd - дескриптор файла, buf - адрес буфера, n - число байт.
Функция возвращает количество реально записанных байт (обычно = n).
fflush (stdout);
выводит текущее содержимое буфера stdout, делая внутренний вызов write (...).
