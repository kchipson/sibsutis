1.12. Терминалы ANSI
Управление экраном осуществляется с помощью esc-последовательностей
(без пробелов, маленькие и большие буквы различаются, код esc = (033)8):
\033[2J стирание экрана
\033[=nF установить цвет символов n = 0 .. 7
\033[=nG установить цвет фона n = 0 .. 7
\033[?25l сделать курсор невидимым
\033[?25h сделать курсор видимым
\033[y;xH    установить курсор в позицию (x,y)
y - номер строки (с 1, 10-ичное число), x - номер столбца (с 1)
стандартный размер экрана - 80 х 25
