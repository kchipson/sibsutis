4.7. Импульсы

Импульс -- короткое сообщение, которое передается без блокировки отправителя.

#include <sys/neutrino.h>
MsgSendPulse (int coid, int prio, int code, int value);

Посылает импульс по соединению coid.

prio   приоритет импульса
code   код импульса (0..127)
value   значение импульса (произвольные 4 байта)

Импульс принимается с помощью функции MsgReceive(...), которая при приеме импульса возвращает 0. Отвечать на импульс не нужно.
