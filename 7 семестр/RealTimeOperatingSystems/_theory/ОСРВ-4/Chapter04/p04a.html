4.4. Передача однокомпонентных сообщений

#include <sys/neutrino.h>
int MsgReceive ( int chid,
void *buf, int bytes, 0);

Принимает сообщение из канала.

chid   идентификатор канала

buf, bytes   адрес и размер буфера для приема сообщения

0   вместо указателя на структуру для записи дополнительной информации о принятом сообщении

Возвращает идентификатор принятого сообщения (rcvid).
