4.5. Передача многокомпонентных сообщений

#include <sys/neutrino.h>
int MsgSendv ( int coid,
const iov_t *siov, int sparts,
const iov_t *riov, int rparts);

Посылает многокомпонентное сообщение и ждет многокомпонентный ответ.

coid   идентификатор соединения

siov, sparts   адрес таблицы IOV и число компонент в сообщении

riov, rparts   адрес таблицы IOV и число компонент в буфере для ответа

Возвращает значение параметра stat из соответствующей фукции передачи ответа.

int MsgReceivev (int chid, const iov_t *riov, int rparts, 0);
MsgReplyv (int rcvid, int stat, const iov_t *riov, int rparts);

Работа и смысл параметров очевидны.
