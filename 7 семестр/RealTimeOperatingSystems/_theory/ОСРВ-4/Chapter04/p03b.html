4.3. Каналы и соединения
Соединение с каналом

Соединения используются для передачи сообщений.

#include <sys/neutrino.h>
int ConnectAttach (int nd, int pid, int chid, int index, int flags);

Создает соединение с каналом.
nd номер узла в сети qnet (magic cookie, 0 -- локальное подключение)
pid номер процесса (0 -- текущий процесс)
chid   идентификатор канала
index = 0, flags = 0.

Возвращает идентификатор соединения (coid), выбираемый из пространства свободных файловых дескрипторов.

Соединение может использоваться всеми нитями процесса для посылки сообщений.

Допускается создание нескольких соединений с одним каналом.

Нити, пославшие сообщения, выстраиваются в очередь к каналу.
