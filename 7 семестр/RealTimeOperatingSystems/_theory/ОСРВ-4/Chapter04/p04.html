4.4. Передача однокомпонентных сообщений

#include <sys/neutrino.h>
int MsgSend ( int coid,
const void *smsg, int sbytes,
void *rmsg, int rbytes);

Посылает сообщение и ждет ответ.

coid   идентификатор соединения

smsg, sbytes   адрес и длина сообщения в байтах

rmsg, rbytes   адрес и размер буфера для приема ответа

Возвращает значение параметра stat из соответствующей фукции передачи ответа MsgReply().
