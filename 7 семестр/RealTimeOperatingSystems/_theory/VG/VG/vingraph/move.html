Move()

Перемещает графический элемент

Прототип:

#include <vingraph.h>

void Move (int ident, int dx, int dy);

Параметры:

ident
Идентификатор графического элемента.
dx
Смещение по горизонтали.
dy
Смещение по вертикали.

Библиотека:

libvg.a

Описание:

Данная функция перемещает графический элемент относительно его текущего положения на указанную величину смещений по горизонтали и вертикали. Смещения могут быть положительными, отрицательными и нулевыми.

Если идентификатор элемента равен нулю, то перемещается окно графического терминала.

Классификация:

VinGraph

Безопасность:
Точка завершения Да
Обработчик прерывания Нет
Обработчик сигнала Да
Нить Да
