Pixel()

Рисует точку (пиксел)

Прототип:

#include <vingraph.h>

int Pixel (int x, int y, int color, int parent);

Варианты:

int Pixel (int x, int y, int color);

int Pixel (int x, int y);

Параметры:

x
Координата x.
y
Координата y.
color
Цвет.
parent
Идентификатор рисунка, в рамках которого создается графический элемент.

Библиотека:

libvg.a

Описание:

Функция рисует точку (пиксел).

Если идентификатор рисунка равен нулю или не указан, то графический элемент создается в рамках всего окна терминала VinGraph. Если цвет не указан, то используется цвет, установленный ранее с помощью SetColor(int color). Для задания цвета можно использовать макрос RGB(r,g,b).

Несмотря на то что Pixel() создает изначально одну точку, позднее эта точка может быть переделана в множество точек с помощью функции EnlargeTo().

Возвращаемое значение:

Идентификатор созданного графического элемента либо -1 в случае ошибки.

Классификация:

VinGraph

Безопасность:
Точка завершения Да
Обработчик прерывания Нет
Обработчик сигнала Да
Нить Да
