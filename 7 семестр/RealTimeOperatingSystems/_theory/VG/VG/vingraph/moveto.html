MoveTo()

Перемещает графический элемент в указанную позицию

Прототип:

#include <vingraph.h>

void MoveTo (int x, int y, int ident);

Варианты:

void MoveTo (int x, int y);

Параметры:

x, y
Координаты точки.
ident
Идентификатор графического элемента.

Библиотека:

libvg.a

Описание:

Данная функция перемещает графический элемент в указанную точку. Точкой привязки элемента является левый верхний угол описывающего прямоугольника.

Если идентификатор элемента опущен или равен нулю, то в указанную (относительно всего экрана) точку перемещается окно графического терминала.

Классификация:

VinGraph

Безопасность:
Точка завершения Да
Обработчик прерывания Нет
Обработчик сигнала Да
Нить Да
