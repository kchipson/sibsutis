Rect()

Рисует прямоугольник

Прототип:

#include <vingraph.h>

int Rect (int x, int y, int w, int h, int roundness, int color, int parent);

Варианты:

int Rect (int x, int y, int w, int h, int roundness, int color);

int Rect (int x, int y, int w, int h, int roundness);

int Rect (int x, int y, int w, int h);

Параметры:

x, y
Координаты левой верхней вершины.
w
Ширина (размер по горизонтали).
h
Высота (размер по вертикали).
roundness
Число пикселов, используемых для закругления углов.
color
Цвет.
parent
Идентификатор рисунка, в рамках которого создается графический элемент.

Библиотека:

libvg.a

Описание:

Функция рисует прямоугольник с прямыми или закругленными углами.

Если идентификатор рисунка равен нулю или не указан, то графический элемент создается в рамках всего окна терминала VinGraph. Если цвет не указан, то используется цвет, установленный ранее с помощью SetColor(int color). Для задания цвета можно использовать макрос RGB(r,g,b).

Закругление roundness является фактически радиусом закругляющей окружности. Если параметр закругления не указан или равен нулю, то рисуется прямоугольник с прямыми углами (см. пример). Отметим, что для многоугольников закругление углов не реализовано. Поэтому, если прямоугольник создается с помощью функции Polygon(), то он может иметь только прямые углы.


Точкой привязки считается точка (x,y).

Возвращаемое значение:

Идентификатор созданного графического элемента либо -1 в случае ошибки.

Классификация:

VinGraph

Безопасность:
Точка завершения Да
Обработчик прерывания Нет
Обработчик сигнала Да
Нить Да
