ConnectGraph()

Подключает прикладной процесс к терминалу VinGraph

Прототип:

#include <vingraph.h>

int ConnectGraph (const char *name);

Параметры:

name
Адрес строки, содержащей имя графического терминала.

Библиотека:

libvg.a

Описание:

Данная функция создает соединение с графическим терминалом VinGraph, при необходимости запуская его. Созданное соединение используется функциями библиотеки VinGraph для посылки сообщений графическому терминалу. Т.о. функция ConnectGraph() должна быть вызвана до вызова любой другой функции библиотеки VinGraph.

Параметр name может быть опущен либо равен нулю. В этом случае используется терминал с именем "VG".

Если терминала с указанным именем не существует, то он запускается. При автоматическом запуске параметры окна терминала соответствуют умолчаниям, заложенным в программу vg. Размер рабочей области и местоположение окна могут быть изменены с помощью мыши. Размер, цвет рабочей области и местоположение окна могут быть изменены программно с помощью функций Fill() и EnlagreTo().

Подключение к терминалу VinGraph наследуется дочерними процессами.

Возвращаемое значение:

0 в случае успеха либо -1 в случае ошибки.

Классификация:

VinGraph

Безопасность:
Точка завершения Да
Обработчик прерывания Нет
Обработчик сигнала Да
Нить Да
