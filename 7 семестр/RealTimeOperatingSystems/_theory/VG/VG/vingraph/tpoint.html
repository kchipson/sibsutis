tPoint

Описание точки

Прототип:

#include <vingraph.h>

struct tPoint
  {
  short int x;
  short int y;
  };

Описание:

Структура tPoint описывает точку с координатами (x,y). Используется рядом функций библиотеки VinGraph при описании типов параметров и возвращаемых значений.

Классификация:

VinGraph

Используется функциями:

Polyline(), Polygon(), EnlargeTo(), GetPos(), GetDim()
