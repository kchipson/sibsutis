GetColor()

Определяет текущий цвет

Прототип:

#include <vingraph.h>

(1) int GetColor ();
(2) int GetColor (int ident);

Параметры:

ident
Идентификатор графического элемента.

Библиотека:

libvg.a

Описание:

Функция (1) возвращает текущую установку цвета, которая используется при рисовании без явного указания цвета. Текущий цвет устанавливается с помощью функции SetColor(int color) локально в каждой нити процесса.

Функция (2) возвращает цвет указанного графического элемента.

Возвращаемое значение:

Значение цвета либо -1 в случае ошибки.

Классификация:

VinGraph

Безопасность:
Точка завершения Да
Обработчик прерывания Нет
Обработчик сигнала Да
Нить Да
