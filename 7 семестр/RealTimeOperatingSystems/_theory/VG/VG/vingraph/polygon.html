Polygon()

Рисует многоугольник

Прототип:

#include <vingraph.h>

int Polygon (const tPoint points [], int npoints, int color, int parent);

Варианты:

int Polygon (const tPoint points [], int npoints, int color);

int Polygon (const tPoint points [], int npoints);

Параметры:

points
Массив, задающий координаты вершин.
npoints
Количество вершин.
color
Цвет.
parent
Идентификатор рисунка, в рамках которого создается графический элемент.

Библиотека:

libvg.a

Описание:

Функция рисует многоугольник с указанными вершинами. Каждая вершина описывается с помощью структуры tPoint.

Если идентификатор рисунка равен нулю или не указан, то графический элемент создается в рамках всего окна терминала VinGraph. Если цвет не указан, то используется цвет, установленный ранее с помощью SetColor(int color). Для задания цвета можно использовать макрос RGB(r,g,b).

Обратим внимание на то, что точкой привязки многоугольника считается самая левая верхняя вершина.

Возвращаемое значение:

Идентификатор созданного графического элемента либо -1 в случае ошибки.

Классификация:

VinGraph

Безопасность:
Точка завершения Да
Обработчик прерывания Нет
Обработчик сигнала Да
Нить Да
