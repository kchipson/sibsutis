Grid()

Рисует пустую таблицу

Прототип:

#include <vingraph.h>

int Grid (int x, int y, int w, int h, int nrows, int ncols, int color, int parent);

Варианты:

int Grid (int x, int y, int w, int h, int nrows, int ncols, int color);

int Grid (int x, int y, int w, int h, int nrows, int ncols);

Параметры:

x, y
Координаты левого верхнего угла таблицы.
w
Ширина таблицы.
h
Высота таблицы.
nrows
Число строк в таблице.
ncols
Число столбцов в таблице.
color
Цвет.
parent
Идентификатор рисунка, в рамках которого создается графический элемент.

Библиотека:

libvg.a

Описание:

Функция рисует пустую (незаполненную) таблицу. Все ячейки таблицы имеют одинаковые размеры.

Если идентификатор рисунка равен нулю или не указан, то графический элемент создается в рамках всего окна терминала VinGraph. Если цвет не указан, то используется цвет, установленный ранее с помощью SetColor(int color). Для задания цвета можно использовать макрос RGB(r,g,b).

Возвращаемое значение:

Идентификатор созданного графического элемента либо -1 в случае ошибки.

Классификация:

VinGraph

Безопасность:
Точка завершения Да
Обработчик прерывания Нет
Обработчик сигнала Да
Нить Да
