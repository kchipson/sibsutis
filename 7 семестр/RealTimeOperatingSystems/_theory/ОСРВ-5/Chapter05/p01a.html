5.1. Простые сигналы
Перехват сигнала 
#include <signal.h>
...
void sigcatch (int signo)
  {
  ...
  exit (signo);
  }

int main ()
  {
  ...
  signal (SIGINT, sigcatch);
  ...
  return 0;
  }

Вместо адреса функции (sigcatch) могут быть указаны константы SIG_DFL и SIG_IGN.

Функция signal() может вызываться многократно, меняя диспозицию сигнала.

В обработчиках сигналов актуально использование процедур языка Си longjmp (siglongjmp).
