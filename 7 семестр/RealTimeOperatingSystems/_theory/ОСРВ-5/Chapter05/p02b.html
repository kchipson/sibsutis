5.2. Сигналы со значением
Эффект наличия нитей

  • маски сигналов
  • "подвешенные" сигналы

sigset_t:   int bits [2];

sigemptyset (sigset_t* ss);
sigadset (sigset_t* ss, int signo);

Установка маски сигналов в текущей нити

pthread_sigmask (int how, const sigset_t* ss, 0);

how = SIG_BLOCK | SIG_UNBLOCK | SIG_SETMASK

Посылка сигнала нити

pthread_kill (int tid, int signo);
